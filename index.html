Interests

My passion is the design and implementation of automatic tools that can boost programmers' productivity to make more reliable and secure code. The goal of my research is to improve the process of verifying and testing complex software systems.

Software

Current Projects

  • SeaHorn : analysis/verification framework for LLVM.
  • Crab : C++ library for static analysis based on abstract interpretation.
  • Clam : static analyzer based on Crab for LLVM.
  • sea-dsa : context-,field-, and array-sensitive pointer analysis for LLVM.
  • PREVAIL : eBPF verifier based on Crab . **PREVAIL in the news**: [1],[2].
  • SmartACE : A verifier of Solidity smart contracts based on SeaHorn. Check out several blog posts here.

Past Projects

  • IKOS : Inference Kernel for Open Static Analyzers. **NASA Group Achievement Award** in 2014.
  • TRACER : interpolation-based symbolic execution tool for verification of C programs.
  • FTCLP: interpolation-based solver for recursive Constrained Horn Clauses.
  • Wrapped Interval Analysis : static analyzer of LLVM bitcode specialized for fixed-width integers.
  • Revenant : string solver.
  • Covenant : semi-decider for intersection of context-free languages.
  • The Ciao Preprocessor : a program analyzer for debugging, verification, and optimization for the Ciao programming language.
